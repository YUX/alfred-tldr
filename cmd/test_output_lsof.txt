[1mlsof[33;0m

Lists open files and the corresponding processes.
Note: Root privileges (or sudo) is required to list files opened by others.

- [32mFind the processes that have a given file open:[33;0m
[31mlsof [34mpath/to/file[31m[33;0m

- [32mFind the process that opened a local internet port:[33;0m
[31mlsof -i :[34mport[31m[33;0m

- [32mOnly output the process ID (PID):[33;0m
[31mlsof -t [34mpath/to/file[31m[33;0m

- [32mList files opened by the given user:[33;0m
[31mlsof -u [34musername[31m[33;0m

- [32mList files opened by the given command or process:[33;0m
[31mlsof -c [34mprocess_or_command_name[31m[33;0m

- [32mList files opened by a specific process, given its PID:[33;0m
[31mlsof -p [34mPID[31m[33;0m

- [32mList open files in a directory:[33;0m
[31mlsof +D [34mpath/to/directory[31m[33;0m
